### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Development

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run develop
```

Your site will be available at `http://localhost:8000`

### Build

To build the site for production:
```bash
npm run build
```
